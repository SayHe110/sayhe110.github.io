    • 关于二维数组修改键值、sql 语句一对多查询的问题

    背景

    今天在撸代码时，遇到一个个人需求，有两个数组（一个是二维， A/B），A 想通过一个键值获取 B（二维） 的值，所以需要修改 B 的键值。
    还有一个个人需求，在写 sql 时，需要写一对多查询，其实使用模型查询更简单，但是时多库查询，所以还是使用了原生 sql 写。

    解决方案

    修改数组键值

    使用 array_column($arr, NULL, 'key_name'); 即可，但是会产生一个新数组

    sql 一对多查询

    使用 GROUP_CONCAT 即可
